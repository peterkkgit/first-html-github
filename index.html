Github Step-by-Step

Creating Repository

  1. Create folder and files on computer - it's ok
  2. Create New Repository on Github
    1. Click "Repositories" > "New", then give a name and check no boxes for files
    2. Left Public
    3. Click "Create Repository"
  3. At this poing, github is very helpful to give you the next steps on screen. Follow these:
  4. Now to Terminal:
    1. To change directory cd /d D:\DRIVE\_SHARED PETER\CODING\HTML-CSS
    2. git init name_of_folder (can type first letter then hit tab to make it easier)
    3. cd name_of_folder
    4. git status - just to check we're in the right folder and files are there - files should show up in red
    5. git add README.md - SKIP THIS
    6. git add . - DO THIS INSTEAD
    7. git status one more time - files should be green now confirming they've been added (to?)
    8. git commit -m "first commit" - WHAT'S THE PURPOSE OF THE MESSAGE?
    9. git remote add origin https://github.com/peterkkgit/first-html-github.git - GITHUB GIVES YOU THIS
    10. git push -u origin master
  5. Voila!

Updating Changes

In both cases, need to be in the root directory ie. name_of_folder>

Check this with a git status - should see files in red that need to be updated

Pushing

Much simpler

  1. git add .
  2. git commit -m "some_message_here" - STILL NOT SURE WHAT THIS IS FOR? IS IT A COMMENT?
  3. git push

Pulling

First, be in the root directory

  1. git pull
